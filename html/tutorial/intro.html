What is Html?

  • Html is the standard markup language for creating Web pages.
  • Html stands for Hyper Text Markup Language
  • Html describes the structure of Web pages using markup
  • Html elements are the building blocks of Html pages
  • Html elements are represented by tags
  • A well formatted Html tag should contain opening and matching closing element e.g. <h1> is the opening element and </h1> is the closing tag.
  • Html content is placed between opening and closing element
  • Html tag with no content is called empty element
  • Browser is a special software that transform Html tags to a content of the page
